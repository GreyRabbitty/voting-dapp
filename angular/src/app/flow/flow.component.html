How the flow of the application works?

1

Verifying the citizen data

User data must be valited by seperated API. If the credentials are matched by the mock data on database than user can go to next step. If user is banned or voted already cannot go for the next step.

2

Validation or verification

At this step user needs to valite to vote, he/she can cancel voting process. This step is necessarry to make sure voting is happening and changing the data on the database, this is the last step of API for voting tracker.

3

Voting

User has to chose one of the candidates, at this point user is already marked as voted, API's checking if the user voted in meantime, app makes transaction through smart contract to make a vote.
